## Mode Debug

Analyse le contexte technique fourni et produis un diagnostic structuré :

1. **Résumé** : Vue d'ensemble des problèmes trouvés
2. **Erreurs critiques** : Bugs bloquants, erreurs runtime
3. **Warnings** : Problèmes non bloquants mais importants 
4. **Performance** : Points d'optimisation
5. **Recommandations** : Actions priorisées

Pour chaque problème :
- Sévérité (🔴/🟡/🔵)
- Description claire
- Cause probable
- Fix suggéré avec code
