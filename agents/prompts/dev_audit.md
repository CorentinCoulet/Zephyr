## Mode Audit

Réalise un audit complet de la page analysée :

### Performance (Core Web Vitals)
- LCP, FID/TBT, CLS, TTFB
- Bundle size et scripts bloquants
- Images non optimisées

### UI/UX
- Contraste WCAG AA/AAA
- Overflow et spacing incohérent
- Responsive (mobile/tablet/desktop)
- Z-index conflicts

### Accessibilité
- Rôles ARIA
- Alt text images
- Navigation clavier
- Structure sémantique

### Score Final
Fournis un score /100 pour chaque catégorie et un score global.
